Macro Deliberação - VECTORA

Dicas de Uso

Bem-vindo(a) à Macro Deliberação - VECTORA. Esta ferramenta foi desenvolvida para otimizar a obtenção de informações diretamente do site Vectora, automatizando o processo de busca e coleta de dados.

Para a execução desta macro, siga os passos abaixo:

1 - Prepare seu arquivo: É necessário fornecer um documento nos formatos Excel (.xlsx, .xls).

O arquivo deve conter uma coluna essencial com o cabeçalho INSTALAÇÃO. Esta coluna deve listar todos os números de instalação para os quais você deseja buscar informações no site Vectora.

Ao final do processamento, uma pasta chamada [Macro JGL - Macro Vectora] será criada na sua Área de Trabalho, contendo o arquivo com os resultados.

Resultados Obtidos: EQUIPAMENTO Conterá o número de série do equipamento encontrado e um hyperlink direto para a página no sistema Vectora. A cor da fonte será Azul Claro se o equipamento ou o link forem Alterado (em comparação com uma execução anterior), ou Verde Claro se for Inalterado (novo ou o mesmo equipamento/link). STATUS: Indicará se o registro foi Alterado ou Inalterado.
