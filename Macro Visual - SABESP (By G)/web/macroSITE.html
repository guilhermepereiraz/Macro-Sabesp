Macro SITE - WFM

Dicas de Uso

Bem-vindo(a) à Macro de SITE - WFM. Esta ferramenta foi desenvolvida para otimizar a obtenção de informações diretamente do site WFM, automatizando o processo de busca e coleta de dados.

Para a execução desta macro, siga os passos abaixo:

1 - Prepare seu arquivo: É necessário fornecer um documento nos formatos Excel (.xlsx, .xls) ou CSV (.csv).

2 - Certifique-se do conteúdo: O arquivo deve conter uma coluna essencial com o cabeçalho OS ou Número OS. Esta coluna deve listar todos os números de Ordens de Serviço para os quais você deseja buscar informações no site WFM.

Ao final do processamento, uma pasta chamada [Macro JGL - Macro SITE] será criada na sua Área de Trabalho, contendo o arquivo com os resultados.
