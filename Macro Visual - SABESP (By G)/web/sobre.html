Sobre a Plataforma Macro JGL

A Plataforma Macro JGL foi desenvolvida com o objetivo de otimizar e automatizar processos operacionais e administrativos, oferecendo ferramentas robustas para gerenciamento de macros e integrações de sistemas. Nosso foco é aprimorar a eficiência e reduzir a carga de trabalho manual para as equipes.

Tecnologias Utilizadas

Esta aplicação foi construída utilizando um conjunto de tecnologias modernas e eficientes:

  • Backend: Python (com `eel` para comunicação entre frontend e backend)
  • Frontend: HTML5, CSS3, JavaScript
  • Banco de Dados: MySQL (para armazenamento local de dados)
  • Design: Font Awesome para ícones, Google Fonts para tipografia
